# Bitrix24 Telegram Bot

Telegram-бот для приёма заявок и ведения сделок в Bitrix24 CRM.

## Возможности

- **Приём заявок** — клиент пишет в Telegram, бот создаёт сделку в CRM
- **Чат клиент↔менеджер** — вся переписка в реальном времени через Telegram
- **Мульти-сделочность** — клиент ведёт несколько сделок, переключается по `#ID`
- **Таймлайн CRM** — каждое сообщение сохраняется в таймлайне сделки Bitrix24
- **Классификация** — автоопределение типа запроса (подбор/расчёт/консультация)

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие |
| `/deals` | Список своих сделок |
| `/help`  | Справка |
| `#ID` | Переключиться на сделку |
| `#ID текст` | Переключиться и отправить сообщение |

## Быстрый старт

1. Создать `.env` файл с токеном:
   ```
   TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
   BITRIX24_WEBHOOK=https://ваш-битрикс.bitrix24.ru/rest/1/токен/
   ```

2. Установить зависимости:
   ```bash
   pip install aiogram requests
   ```

3. Запустить:
   ```bash
   python tgbot_leads.py
   ```

## Файлы проекта

| Файл | Назначение |
|------|-----------|
| `tgbot_leads.py` | Основной бот |
| `process_leads.py` | Cron-обработчик: классификация, контакты, компании |
| `email_tracker.py` | Отслеживание входящей почты |
| `start_bot.bat` | Быстрый запуск на Windows |
